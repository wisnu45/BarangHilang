# Owncart - An Ecommerce Solution

## Technologies Used

* CodeIgniter Framework
* HTML - CSS 
* Javascript
* jQuery
* Ajax
* Bootstrap

## Functionalities

* Session Management
* Cart Management
* Admin Panel
* Login Management
* Cookies Management
* Real - time Product Filtering using Ajax
* Payment Mode Available (Need to set paypal account)
* Check Pin Code ( Add Google API key First)
* Responsive Design
